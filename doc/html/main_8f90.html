Detailed Description

This file contains the main subroutines for the example problem of PaScaL_TDMA.

The target example problem is the three-dimensional time-dependent heat conduction problem in a unit cube domain applied with the boundary conditions of vertically constant temperature and horizontally periodic boundaries.

)

Write the values of field variable in a output file.

It writes the values of field variables with two methods, single task IO and post assembly IO. In single task I/O, each process writes the values in its own file. In post assembly IO, each process send data to a master process and the master process write the obtained data in a single file.

Parameters
myrankRank ID in MPI_COMM_WORLD
nprocsNumber of MPI process in MPI_COMM_WORLD
theta_subField variable to write
